Funzione principale.

Tramite una sequenza di dialogi permette all'utilizzatore di effettuare le operazioni sulle operazioni.
Nell'ordine:

  • crea il calendario;
  • riceve in input le prenotazioni;
  • visualizza il calendario;
  • elimina le prenotazioni completate e visualizza le rimanenti;
  • modifica le prenotazioni e le visualizza;
  • ordina le prenotazioni e le visualizza;
  • visualizza le prenotazioni attuali;
  • visualizza le prenotazioni in data specificata.
